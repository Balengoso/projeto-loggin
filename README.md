# Projeto - Tela de Login

### UNDER CONSTRUCTION

#### Descrição

Projeto desenvolvido na disciplina de Programação Front-End na Unicesumar, campus de Londrina. Durante as aulas, o professor [Leonardo Rocha](https://github.com/leonardossrocha) apresentou os conceitos iniciais de framework, utilizando o Bootstrap como exemplo prático, além de introduzir o uso do Git para controle de versão.

Neste projeto estou aprendendo a:

* Entender o que é e como utilizar um framework.
* Utilizar componentes e exemplos disponíveis no Bootstrap.
* Trabalhar com controle de versão usando Git.
* Executar comandos básicos do Git, como:

  * `git status`
  * `git add`
  * `git commit -m`
  * `git push`
* Criar e organizar a documentação no README.
* Fazer comentários no código para melhorar a organização e entendimento.
* Desenvolver uma página de FAQ (Perguntas Frequentes).
* Realizar tradução de página como prática de desenvolvimento.

Este projeto faz parte do meu processo de aprendizado e evolução na área de desenvolvimento Front-End.

### Funcionalidades
- Colocar Email
- Colocar Senha
- Marcar para continuar logado
- Entrar em sua conta
- FAQ

### Ferramentas utilizadas

- HTML5
- CSS3
- Bootstrap
- Git
